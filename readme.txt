== Description ==

A full fledged wp e-commerce product search widget with grid view. NO GOLD CART REQUIRED. This search widget was orginally developed for our site <a href=http://www.salespagehost.com target=#new>http://www.salespagehost.com</a>. to give a grid view product search results page without the gold cart plugin. 

Languages Supported: 

* English
* German
* French

== Installation ==

The plugin requires wp e-commerce plugin.

1. Upload the `wp-e-commerce-product-search-widget` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to a dynamic sidebar and customise the features required.
